# Orio API - README

## Installation

1. Copier le fichier .env.example vers .env
```bash
cp .env.example .env
```

2. Installer les dépendances
```bash
composer install
```

3. Générer la clé d'application
```bash
php artisan key:generate
```

4. Lancer les migrations
```bash
php artisan migrate
```
